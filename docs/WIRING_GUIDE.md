# 🔌 2단계: 하드웨어 선 연결 가이드

> ⚠️ **중요: 반드시 전원이 꺼진 상태에서 연결하세요!**

## 라즈베리파이5 GPIO 핀 배치도

```
                    ┌──────────────────────┐
                    │  라즈베리파이5 (윗면)  │
    3.3V      (1)  ●  ●  (2)  5V ◄── SPS30 VDD
 SDA/GPIO2   (3)  ●  ●  (4)  5V ◄── LCD VCC
 SCL/GPIO3   (5)  ●  ●  (6)  GND ◄── SPS30 SEL
              (7)  ●  ●  (8)
       GND    (9)  ●  ●  (10)
             (11)  ●  ●  (12)
             (13)  ●  ●  (14) GND ◄── LCD GND
             ... (이하 생략)
                    └──────────────────────┘
```

## SPS30 센서 연결 (5가닥)

| SPS30 핀 | 기능 | 라즈베리파이 핀 |
|----------|------|----------------|
| 1 (VDD) | 전원 5V | Pin 2 (5V) |
| 2 (SDA) | 데이터 | Pin 3 (SDA) ★공유 |
| 3 (SCL) | 클럭 | Pin 5 (SCL) ★공유 |
| 4 (SEL) | 모드선택 | Pin 6 (GND) ⚠️필수! |
| 5 (GND) | 그라운드 | Pin 9 (GND) |

> ⚠️ **SEL 핀(4번)을 GND에 반드시 연결해야 I2C 모드로 작동합니다!**

## LCD 1602 I2C 연결 (4가닥)

| LCD 핀 | 기능 | 라즈베리파이 핀 |
|--------|------|----------------|
| VCC | 전원 5V | Pin 4 (5V) |
| GND | 그라운드 | Pin 14 (GND) |
| SDA | 데이터 | Pin 3 (SDA) ★공유 |
| SCL | 클럭 | Pin 5 (SCL) ★공유 |

## ★ SDA/SCL 핀 공유

SPS30과 LCD 모두 3번(SDA), 5번(SCL)에 연결해야 합니다.

**방법 1: 브레드보드** (추천) - 한 줄에 두 선을 꽂으면 됩니다.

**방법 2: 전선 꼬기** - 두 선 끝을 꼬아서 하나의 핀에 함께 꽂습니다.

## 연결 확인

```bash
i2cdetect -y 1
```

- `27` (또는 `3F`) = LCD → config.py에서 설정
- `69` = SPS30 센서

**두 숫자가 보이면 성공! 🎉**
